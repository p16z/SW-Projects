## 1ο Παραδοτέο - Εισαγωγή
Ένας από τους στόχους  του μαθήματος είναι να τόσο η συνεργασία με άλλα άτομα μέσω της ομάδας και συγκεκριμένα μέσω της εκμάθησης της ανάπτυξης λογισμικού όσο και η εξοικείωση και διατήρηση του ομαδικού πνεύματος για την διευκόλυνση των συνθήκων εργασίας. Παράλληλα, ένα από τα μεγάλα οφέλη που μπορεί να αποκτήσει κάνεις μετά από  την ενασχόληση με την ανάπτυξη του λογισμικού είναι η απόκτηση εμπειριών αλλά και η ανακάλυψη νέων δεξιοτήτων. Πολύ σημαντική είναι η εξοικείωση με το linux αλλά και το github που η γνώση τους είναι αναγκαίο εργαλείο για ένα άτομο που δραστηριοποιείται στον χώρο της ανάπτυξη λογισμικού. Αυτό το εξάμηνο είναι μια καλή ευκαιρία για εξερεύνηση των λειτουργικά χωρίς systemd ως το init τους, για να παρατηρήσουμε πόση μεγάλη διαφορά έχουν από το systemd σε performance. Μετά από κάποιες προσπάθειες εγκατάστασης διάφορων άλλων λειτουργικών συστημάτων κατέληξα στο FreeBSD. Η ιδέα του λειτουργικού χωρίς systemd στην αρχή δεν με ενθουσίασε ιδιαίτερα καθώς είχα συνηθίσει να χρησιμοποιώ Windows. Μετά την εγκατάσταση του FreeBSD που διάλεξα μου φάνηκε αρκετά ενδιαφέρον η χρήση του, o τρόπος λειτουργίας και οι δυνατότητες που έχει σαν λογισμικό.

Επίσης, μέσα από τα βίντεο υπάρχουν και διαφορά πολύτιμα μαθήματα διαφόρων ειδών. Ένα από αυτά  είναι ότι πρέπει να αλλάξουμε την νοοτροπία ότι γνωρίζουμε τα πάντα και να παρατηρήσουμε και άλλες οπτικές καταστάσεων. Σε αυτό συμβάλλει βεβαίως συμβάλλουν διάφορους φορείς όπως για παράδειγμα το εκπαιδευτικό σύστημα. Με το να επαναλαμβάνουμε μεθόδους και νοοτροπίες, δεν υποστηρίζουμε καθόλου τους νέους επιστήμονες. Το εκπαιδευτικό σύστημα μας ωθεί κατά κάποιο τρόπο στην ίδια επαναλαμβανόμενη διαδικασία και σκέψη, δίνοντας σημασία μόνο στα προγραμματιστικά περιβάλλοντα. Ωστόσο,  έχοντας λάβει υπόψιν όσα ειπώθηκαν παραπάνω, υπάρχει σημαντικό όφελος για όποιον ασχοληθεί με το συγκεκριμένο μάθημα.

<img src="https://github.com/p16z/assets/raw/main/A/pdf-canvas.png" alt="drawing" width="500"/> <img src="https://github.com/p16z/assets/raw/main/A/pdf-canva2s.png" alt="drawing" width="500"/>

Το PdfLatex είναι ένα εργαλείο που μετατρέπει πηγές Latex σε PDF. Αυτό είναι ιδιαίτερα σημαντικό για τους ερευνητές, καθώς το χρησιμοποιούν για να δημοσιεύσουν τα ευρήματά τους. Θα μπορούσε να εγκατασταθεί πολύ εύκολα χρησιμοποιώντας τερματικό Linux, αν και αυτό φαίνεται ενοχλητικό έργο στα Windows. Οι εντολές εγκατάστασης δίνονται παρακάτω.

* Εγκατάσταση του TexLive base 

```
sudo apt-get install texlive-latex-base
```

* Για να αποφυγή σφάλματων χρησιμοποιούμε pdflatex που προσφέρει περισσότερες γραμματοσειρές

```
sudo apt-get install texlive-fonts-recommended
sudo apt-get install texlive-fonts-extra
```

* Εγκατάσταση extra packages,

```
sudo apt-get install texlive-latex-extra
```

Αφού είχαν εγκατασταθεί τα παραπάνω τότε μπόρεσα να δημιουργήσω αρχεία PDF από πηγές λατέξ χρησιμοποιώντας το PdfLatex.

```
pdflatex cv.tex
```

## 9o Παραδοτέο - Άσκηση γραμμής εντολών
Στο αυτό το παραδοτέο χρειάστηκε να πραγματοποιήσουμε μια άσκηση γραμμής εντολών μέσω τερματικού. Προσωπικά επέλεξα να ασχοληθώ με την εργασία «Send notification to desktop» μέσα από το ntfy. Μέσω αυτής έχουμε τη δυνατότητα να στείλουμε ειδοποιήσεις στην επιφάνεια εργασίας ή στο κινητό. Στην συνέχεια απεικονίζεται η διαδικασία μέσω της οποίας πραγματοποιείται η αποστολή notification όταν τελειώνει μια εργασία των 7 δευτερολέπτων. Επειδή αντιμετώπισα προβλήματα με την εγκατάσταση της εντολής ntfy στο FreeBSD σκέφτηκα να πραγματοποιήσω την άσκηση στα Ubuntu αλλά επειδή γνώριζα ότι θα βαθμολογηθεί αρνητικά λόγω του ότι τα Ubuntu διαθέτουν system προσπάθησα να ψάξω περισσότερες πληροφορίες για την εγκατάσταση των πακέτων στο FreeBSD. Τελικά κατάφερα να εγκαταστήσω το ntfy αλλά ένα άλλο πρόβλημα που αντιμετώπισα είναι ότι δεν εμφανιζόντουσαν τα notification στην οθόνη.

## 12ο Παραδοτέο - Τελική αναφορά & Συμπεράσματα
Με την ολοκλήρωση του μαθήματος και των παραδοτέων οι γνώσεις και οι απόψεις μου για την ανάπτυξη του λογισμικού έχουν εμπλουτιστεί. Αρχικά, με την εγκατάσταση καινούργιων λογισμικών διαπίστωσα τα λογισμικά που είναι δημοφιλές στην αγορά στις μέρες μας περιορίζουν τον χρήστη σε αντίθεση με άλλα Linux distributions. Παράλληλα, μέσω των γραμμών εντολών κατάφερα να εξοικειωθώ με το καινούργιά συστήματα που εγκατέστησα και οι δεξιότητες και γνώσεις μου πάνω στο GitHub έχουν βελτιωθεί αναμφισβήτητα. Επίσης, παρατήρησα πόσο σημαντικό είναι το κάθε μέλος να κατανοεί τον ρόλο του στην ομάδα με αποτέλεσμα να μπορεί να αποδώσει καλυτέρα και να έχει μια καλή απόδοση όσον αφορά την σύνθεση των εργασιών.
